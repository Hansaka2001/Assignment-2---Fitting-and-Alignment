## Overview
This repository contains the Python scripts, Jupyter Notebook, and output results for Assignment 02 of the IN4640 Machine Vision module. The project explores core mathematical techniques in computer vision, including Total Least Squares (TLS), RANSAC, planar dimensional analysis, and perspective transformations (Homography).

## Repository Structure

### Code Files
* **`Q01.ipynb`**: A Jupyter Notebook implementing Total Least Squares (TLS) using Eigenvalue Decomposition and a custom-built RANSAC algorithm from scratch to extract multiple linear models from noisy data.
* **`Q02.py`**: A Python script that utilizes OpenCV contour detection (with Otsu's thresholding) and pinhole camera geometry to automatically calculate the physical dimensions (in millimeters) of a pair of earrings, plotted onto a physical millimeter grid.
* **`Q03.py`**: An interactive Python script that computes a planar homography matrix based on user-selected coordinates to realistically superimpose a 2D country flag onto a 3D cricket turf using alpha-blending and masking.

###  Outputs
* Included output images demonstrate the graphical results of the algorithms (`.png` and `.jpg` files generated by the scripts).
